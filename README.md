🛒Sistema de Inventario "La Tiendita" — Proyecto Final C#

🚀 Instalación y Ejecución

Sigue estos pasos para ejecutar el proyecto localmente:
1. Clonar o descargar

Descarga el archivo Program.cs o clona este repositorio.
2. Configurar el proyecto

Si usas la terminal, crea un nuevo proyecto de consola:
Bash

dotnet new console -n LaTiendita
cd LaTiendita

3. Compilar y Ejecutar

Copia el código en tu archivo Program.cs y ejecuta:
Bash

dotnet run
